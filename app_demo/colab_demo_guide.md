##### Steps to use Colab T4 runtime to run Streamlit App:
1. Ensure that `chromadb/`, `movie_titles.txt`, and `movie_title_embeddings.npy` are located in your MyDrive
2. Upload app_demo.ipynb to Colab
3. Connect to T4 runtime
4. Upload all three .py scripts into the runtime storage
5. Run all cells in the app_demo.ipynb
6. Click the link in the ongoing cell output of the last cell to open the Streamlit Web App
